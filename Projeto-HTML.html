Contextualização do Projeto:

Site desenvolvido para praticar a digitação em diferentes níveis de dificuldade, contabilizando a quantidade de palavras corretas, erradas e o tempo restante da prática.

Arquivos utilizados:

  • index.html
  • estilos.css
  • gerenciador.js

É definido a função de contagem de tempo para a digitação. Assim que o usuário começa a digitar, o cronômetro inicia a contagem, sendo o tempo restante de 60 segundos e vai diminuindo 1 a cada 1000 milissegundos. Após o término do tempo restante ele chama a função de verificar resultado automaticamente.

function iniciarContadorTempo() {
    if (intervaloTempo) clearInterval(intervaloTempo);

    tempoRestante = 60;

    document.getElementById('contadorTempo').textContent = tempoRestante;

    intervaloTempo = setInterval(() => {

        tempoRestante--;

        document.getElementById('contadorTempo').textContent = tempoRestante;

        if (tempoRestante <= 0) {
            clearInterval(intervaloTempo);
            verificarResultado();
        }
    }, 1000);
}

É definida a função de verificação de resultado. Assim que o usuário termina de digitar ou então o cronômetro chega a 0, o código calcula a precisão da digitação do usuário. O usuário só será aprovado se seu resultado for acima de 80%.

function verificarResultado() {
    const porcentagemCerta = contadorCerto / textoAtual.length;

    if (porcentagemCerta >= 0.8) {
        alert('Parabéns! Você foi aprovado.');
    } else {
        alert('Tente novamente!')
    }

    resetar();
}

É definida a função de resetar para que o usuário possa praticar mais vezes, limpando o texto, bloqueando a entrada de dados, redefinindo todos os contadores e removendo as cores certas e erradas do texto.

function resetar() {
    const elementoEntrada = document.getElementById('entrada');
    elementoEntrada.value = '';

    elementoEntrada.disabled = true

    clearInterval(intervaloTempo);

    document.getElementById('contadorCerto').textContent = '0';
    
    document.getElementById('contadorErrado').textContent = '0';

    document.getElementById('contadorTempo').textContent = '60';

    tempoRestante = 60;

    contadorCerto = 0;

    contadorErrado = 0;

    [...document.getElementById('texto').children].forEach(span => {
        span.classList.remove('certo', 'errado');
    });
}

É definida a função de mudar de nível/dificuldade de digitação. Quando o usuário seleciona um nível de dificuldade, as informações de contagem e estatísticas são resetadas, o texto a ser digitado é trocado para o nível correspondido e o campo de digitação é desbloqueado para que o usuário possa começar a treinar a digitação.

function mudarNivel(nivel) {
    resetar();

    textoAtual = textos[nivel - 1];

    const elementoTexto = document.getElementById('texto');

    elementoTexto.innerHTML = textoAtual.split('').map(char => `<span>${char}</span>`).join('');

    document.getElementById('entrada').disabled = false;
}

É definido o evento que juntará todas as funções apresentadas e fará com que o curso de digitar funcione por completo. O usuário irá abrir a tela, selecionará a dificuldade e começar a digitar. A partir do momento que o usuário começar a digitar será iniciado o tempo e a contagem de letras certas e erradas, comparando o que o usuário está digitando com o texto armazenado dentro da lista. Se a letra for igual irá ficar verde e se estiver errado ficará vermelha.

Quando o usuário terminar de digitar ou quanto o contator de tempo tiver terminado, o código irá apresentar as estatísticas e quanto tempo sobrou. Se as estatísticas estiverem menores que 80%, o sistema irá reprovar o usuário.

document.getElementById('entrada').addEventListener('input', function() {
    if (tempoRestante === 60) iniciarContadorTempo();

        const entradaTexto = this.value;

    if (entradaTexto.length > textoAtual.length) {
        this.value = entradaTexto.substring(0, textoAtual.length);

        return
    }

    contadorErrado = 0;

    [...document.getElementById('texto').children].forEach((span, index) => {
        if (index < entradaTexto.length) {
            if (entradaTexto[index] === span.textContent) {
                span.classList.add('certo');

                span.classList.remove('errado');

                contadorCerto++;
            } else {
                span.classList.add('errado');

                span.classList.remove('certo');

                contadorErrado++;
            }
        } else {
            span.classList.remove('certo', 'errado');
        }
    });

    document.getElementById('contadorCerto').textContent = contadorCerto;

    document.getElementById('contadorErrado').textContent = contadorErrado;

    if (entradaTexto.length === textoAtual.length) {
        verificarResultado();
    }
})
